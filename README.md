### Preparación del HTML y CSS
Bueno, lo primero es admitir que lo suyo me ha costado intentar hacer el codigo por encima del anterior, asi que opté por hacer un nuevo archivo y copiar el codigo del anterior, para luego ir editando lo que se me pide, pero bueno, lo importante es aprender y divertirse, asi que vamos a ello.

He añadido varias etiquetas semánticas para estructurar el contenido de la página, como;
 `<header>`, `<main>`, `<section>`, `<article>` `<ol>` `<ul>`   
Esto ayuda a mejorar la accesibilidad y la organización del contenido.

Además de los elementos específicos para ayudar a jerarquizar.
El atajo del teclado 'Ctrl + Alt + F' me ha ayudado a formatear el código de manera consistente, lo que mejora la legibilidad.

Y gracias a esto me di cuenta que no edité el `README.md` del ejercicio anterior, así que aprovecho para editarlo ahora y explicar un poco lo que he aprendido con este ejercicio.

Gracias a las exposiciones he ido añadiendo poco a poco cosas que no sabía como por ejemplo el `box-shadow` para darle un poco de profundidad a las tarjetas, o el `text-shadow` para darle un poco de vida a los textos, también he aprendido a usar `flexbox` para centrar los elementos y hacer que se vean bien en diferentes tamaños de pantalla.
