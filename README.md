### Preparación del HTML ###
Bueno, lo primero es admitir que lo suyo me ha costado intentar hacer el codigo por encima del anterior, asi que opté por hacer un nuevo archivo y copiar el codigo del anterior, para luego ir editando lo que se me pide, pero bueno, lo importante es aprender y divertirse, asi que vamos a ello.

Además de los elementos específicos para ayudar a jerarquizar.
El atajo del teclado 'Ctrl + Alt + F' me ha ayudado a formatear el código de manera consistente, lo que mejora la legibilidad.
